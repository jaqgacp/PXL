# PXL Typography Standard

Status: OFFICIAL PLATFORM TYPOGRAPHY STANDARD

PXL uses one enterprise UI font stack:

`Inter, Segoe UI, Roboto, Helvetica Neue, Arial, sans-serif`

Do not mix display, monospace, or decorative UI fonts in normal workspace text. Numeric values may use tabular numbers, but they must keep the same UI font.

## Scale

| Role | Size | Weight | Notes |
| --- | ---: | --- | --- |
| Workspace Title / Document Number | 20px | 600 | Header identity |
| Customer / Vendor Name | 16px | 600 | Clickable brand-colored identity |
| Section Headers | 14px | 600 | Uppercase, slight letter spacing |
| Tab Labels | 13px | 500 | Text-only |
| Field Labels | 12px | 500 | Clear but secondary |
| Body Text | 13px | 400 | Default ERP text |
| Table Content | 13px | 400 | Compact but readable |
| Captions | 11px | 400 | Supporting text only |
| Status Badges | 12px | 500 | Semantic state |
| Totals | 18px | 700 | Header totals and important summaries |

## Rules

- Use tabular numbers for amounts, quantities, rates, and totals.
- Do not use negative letter spacing.
- Use uppercase only for section headers, table headers, and compact labels.
- Avoid arbitrary font sizes outside the scale.
