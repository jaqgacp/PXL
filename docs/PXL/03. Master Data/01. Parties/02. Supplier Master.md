# PXL ERP Blueprint: Supplier Master

## Module Overview
The Supplier Master (Vendor Master) is the foundational AP sub-ledger record. Every Purchase Invoice, Payment Voucher, Debit Memo, and Credit Memo on the purchasing side must be linked to a supplier record. In the Philippine context, the supplier record stores BIR-critical data: TIN, VAT status, and the default EWT code that determines withholding tax computations. The EWT defaults on this record drive the MAP (Monthly Alphalist of Payees) and BIR Form 2307 issuance. Missing or incorrect EWT assignments are the most frequent audit finding in BIR MAP submissions.

## Supabase Database Architecture

### Table 1: `suppliers`

**Critical Database Rules:**
1. Composite unique constraint on `(company_id, supplier_code)`.
2. `default_ewt_code_id` is a required FK. If left null, the system must block Purchase Invoice posting and demand EWT assignment. This is the core driver of MAP/2307 compliance.
3. `ewt_exempt = true` requires `ewt_exemption_cert_url` to be non-null (enforced at application layer).

| Column Name | Data Type | Rules | Description |
| :--- | :--- | :--- | :--- |
| `id` | UUID | Primary Key | System identifier. |
| `company_id` | UUID | Required, Foreign Key | Links to `companies.id`. Tenant isolation anchor. |
| `supplier_code` | Text | Required | Unique per company. Auto-generated or manual. |
| `supplier_name` | Text | Required | Registered name per BIR Form 2303. |
| `supplier_type` | Text | Required | `individual`, `corporation`, `government`, `non_resident`. |
| `tin` | Text | Required | Philippine TIN (`NNN-NNN-NNN-VVVV`). Required for 2307 and MAP generation. |
| `tin_branch_code` | Text | Nullable, Default `000` | 3-digit branch code of TIN. |
| `vat_status` | Text | Required | `vat_registered`, `non_vat`, `exempt`. |
| `default_ewt_code_id` | UUID | Nullable, Foreign Key | Links to `ewt_codes.id`. The default EWT code for this supplier. Auto-fills on Purchase Invoice. |
| `ewt_exempt` | Boolean | Required, Default `false` | If `true`, EWT is not withheld. Requires certificate. |
| `ewt_exemption_cert_url` | Text | Nullable | Supabase Storage URL for the exemption certificate. Required if `ewt_exempt = true`. |
| `registered_address` | Text | Required | Full BIR-registered address. |
| `city` | Text | Required | City or municipality. |
| `province` | Text | Required | Province. |
| `zip_code` | Text | Required | Postal code. |
| `rdo_id` | UUID | Nullable, Foreign Key | Links to `ref_rdo_codes.id`. |
| `contact_person` | Text | Nullable | |
| `contact_number` | Text | Nullable | |
| `email_address` | Text | Nullable | |
| `ap_account_id` | UUID | Required, Foreign Key | Links to `chart_of_accounts.id`. AP control account. |
| `payment_terms_id` | UUID | Required, Foreign Key | Links to `payment_terms.id`. |
| `currency_code` | Text | Required, Default `PHP` | Default transaction currency. |
| `default_input_vat_id` | UUID | Nullable, Foreign Key | Links to `vat_codes.id`. Auto-fills Input VAT on Purchase Invoice. |
| `is_active` | Boolean | Required, Default `true` | Active status. |
| `created_by` | UUID | Required, Foreign Key | Links to `users.id`. |
| `updated_by` | UUID | Nullable, Foreign Key | Links to `users.id`. |
| `created_at` | Timestamp | Auto | Creation timestamp. |
| `updated_at` | Timestamp | Auto | Last update timestamp. |

**Composite Unique Constraint:** `UNIQUE(company_id, supplier_code)`
