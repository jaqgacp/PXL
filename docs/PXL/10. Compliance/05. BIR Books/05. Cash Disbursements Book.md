## 1. Module Overview
The Cash Disbursements Book is a mandatory accounting record required by the BIR. This book dynamically pulls line-by-line transaction data from the General Ledger and respective sub-ledgers, formatting it into the BIR-prescribed columnar layout. Data is heavily dependent on accurate Setup configurations and posted Journal Entries.

## 2. Dashboard UI
### The Action Bar
* **Search:** Find specific records by reference or period.
* **Filter:** Filter by Date Range, Status, or Type.
* **Export:** Download to Excel/CSV for BIR submission or offline review.
* **Import:** Import external adjustments or offline templates.
* **Generate:** Pull latest data from Accounting ledgers.

### The Data List
Displays generated records with:
* Reference / Period
* Total Amounts
* Status (Draft, Final, Filed)
* Actions (View, Edit, Print)

## 4. Supabase Database Architecture
Data is structured into Header and Line-Item tables to maintain proper normalization and support BIR CAS standards.

**Critical Database Rule:** A Unique constraint applies to `company_id` + `period` to prevent duplicate reports for the same period.

### Table 1: `compliance_cash_disbursements_book_headers`
| Column Name | Data Type | Rules | What it stores |
| :--- | :--- | :--- | :--- |
| `id` | UUID | Primary Key | System ID. |
| `company_id` | UUID | Required, Foreign Key | Links to Setup module. |
| `period` | Date | Required | Reporting period. |
| `status` | Text | Required | Draft, Final. |
| `created_by` | UUID | Required, Foreign Key | User ID. |
| `updated_by` | UUID | Nullable, Foreign Key | User ID. |
| `created_at` | Timestamp | Auto | Date created. |
| `updated_at` | Timestamp | Auto | Date updated. |

**Critical Database Rule:** A Unique constraint applies to `header_id` + `transaction_id` to prevent duplicate line items.

### Table 2: `compliance_cash_disbursements_book_lines`
| Column Name | Data Type | Rules | What it stores |
| :--- | :--- | :--- | :--- |
| `id` | UUID | Primary Key | System ID. |
| `header_id` | UUID | Required, Foreign Key | Links to `compliance_cash_disbursements_book_headers`. |
| `transaction_id` | UUID | Required, Foreign Key | Direct link to Accounting Ledger entry. |
| `amount` | Numeric | Required | Item amount. |
| `created_by` | UUID | Required, Foreign Key | User ID. |
| `updated_by` | UUID | Nullable, Foreign Key | User ID. |
| `created_at` | Timestamp | Auto | Date created. |
| `updated_at` | Timestamp | Auto | Date updated. |
