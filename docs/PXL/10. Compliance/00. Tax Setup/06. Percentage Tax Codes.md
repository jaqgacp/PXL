## Module Overview
Percentage Tax Codes apply to Non-VAT registered businesses or specific industries (like transport or banks) whose gross receipts are subject to a flat percentage (e.g., 3% under Section 116). This maps to BIR Form 2551Q.

## The Supabase Database Architecture

**Critical Database Rule:** Ensure appropriate unique constraints are applied to prevent duplicate entries.

### Table 1: `percentage_tax_codes`
| Column Name | Data Type | Rules | What it stores |
| :--- | :--- | :--- | :--- |
| `id` | UUID | Primary Key | System ID. |
| `tax_code_id` | UUID | Required, Foreign Key | Links to master `tax_codes.id`. |
| `atc_id` | UUID | Required, Foreign Key | Links to `atc_codes.id`. |
| `form_type` | Text | Required | Usually `2551Q`. |
| `created_by` | UUID | Required, Foreign Key | Links to `users.id`. |
| `updated_by` | UUID | Nullable, Foreign Key | Links to `users.id`. |
| `created_at` | Timestamp | Auto | Creation date. |
| `updated_at` | Timestamp | Auto | Update date. |
