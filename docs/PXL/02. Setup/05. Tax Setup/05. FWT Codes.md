# PXL ERP Blueprint: FWT Codes

## Module Overview
Final Withholding Tax (FWT) Codes cover taxes withheld that are considered final, meaning the payee no longer needs to declare them as part of gross income for regular ITR. Examples include dividends and royalties. This setup supports the generation of BIR Form 1601-FQ.

## What You Will See on the Screen (The Dashboard)
Registry of FWT parameters.

* **Search Bar:** Code.
* **Export Button:** Download list.

## The Supabase Database Architecture (For Lovable)

**Critical Database Rule for Lovable:** Ensure appropriate unique constraints are applied to prevent duplicate entries.

### Table 1: `fwt_codes`
| Column Name | Data Type | Rules | What it stores |
| :--- | :--- | :--- | :--- |
| `id` | UUID | Primary Key | System ID. |
| `tax_code_id` | UUID | Required, Foreign Key | Links to master `tax_codes.id`. |
| `atc_id` | UUID | Required, Foreign Key | Links to `atc_codes.id`. |
| `form_type` | Text | Required | Usually `1601FQ`. |
| `created_by` | UUID | Required, Foreign Key | Links to `users.id`. |
| `updated_by` | UUID | Nullable, Foreign Key | Links to `users.id`. |
| `created_at` | Timestamp | Auto | Creation date. |
| `updated_at` | Timestamp | Auto | Update date. |
