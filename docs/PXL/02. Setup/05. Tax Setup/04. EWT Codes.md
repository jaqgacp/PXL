# PXL ERP Blueprint: EWT Codes

## Module Overview
Expanded Withholding Tax (EWT) Codes are critical for generating BIR Form 2307 (Certificate of Creditable Tax Withheld at Source) and compiling the MAP (Monthly Alphalist of Payees) and QAP (Quarterly Alphalist of Payees). This module links standard tax deduction percentages to specific Alphanumeric Tax Codes (ATC) mandated by the BIR for services, rentals, goods purchases, and contractor payments.

Each EWT Code record must be a fully self-contained, queryable entity with its own code, description, and rate — so that Purchase Invoice lines, Supplier records, and MAP generation can all reference and display the EWT information without requiring deep JOINs through parent tables.

## The Supabase Database Architecture

### Table 1: `ewt_codes`

**Critical Database Rules:**
1. This table must store `ewt_code`, `description`, and `rate` directly — not just a reference to `tax_codes`. The EWT code must be queryable on its own for MAP generation, Supplier Master dropdowns, and Purchase Invoice line auto-fill without requiring multiple JOINs.
2. Composite unique constraint on `(atc_id, rate)` — prevents creating two identical EWT codes for the same ATC and rate.
3. `company_id` is required for multi-tenant isolation. Different companies may configure different active EWT codes even if the BIR ATCs are global.

| Column Name | Data Type | Rules | Description |
| :--- | :--- | :--- | :--- |
| `id` | UUID | Primary Key | System identifier. |
| `company_id` | UUID | Required, Foreign Key | Links to `companies.id`. Tenant isolation anchor. |
| `tax_code_id` | UUID | Required, Foreign Key | Links to master `tax_codes.id`. Parent GL and type classification. |
| `ewt_code` | Text | Required | Unique internal code (e.g., `EWT-WC158-10`). |
| `description` | Text | Required | Full BIR-aligned description (e.g., "Professional Fees - 10%"). Printed on BIR Form 2307. |
| `atc_id` | UUID | Required, Foreign Key | Links to `atc_codes.id`. The BIR ATC. |
| `rate` | Numeric(5,2) | Required | The withholding rate percentage (e.g., `10.00`). Stored directly for fast computation. |
| `form_type` | Text | Required | BIR remittance form: `1601EQ` (EWT), `1601FQ` (FWT), `2550M` (VAT-registered PT). |
| `is_active` | Boolean | Default: `true` | Status toggle. |
| `created_by` | UUID | Required, Foreign Key | Links to `users.id`. |
| `updated_by` | UUID | Nullable, Foreign Key | Links to `users.id`. |
| `created_at` | Timestamp | Auto | Creation date. |
| `updated_at` | Timestamp | Auto | Update date. |

**Composite Unique Constraints:**
- `UNIQUE(company_id, ewt_code)`
- `UNIQUE(company_id, atc_id, rate)` — prevents duplicate EWT setups for the same ATC + rate combination.
