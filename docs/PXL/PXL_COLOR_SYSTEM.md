# PXL Color System

Status: OFFICIAL PLATFORM COLOR STANDARD

Color communicates hierarchy, brand, and state. It is not decoration.

## Semantic Colors

| Purpose | Standard |
| --- | --- |
| Primary Navigation | Dark navy |
| Brand Accent | Burgundy |
| Success | Professional green |
| Warning | Professional amber |
| Error / Danger | Professional red |
| Neutral | Gray scale |
| Primary Text | `#1F2937` |
| Secondary Text | `#4B5563` |
| Muted Text | `#6B7280` |

## Transaction Families

| Family | Header / Tab Tint |
| --- | --- |
| Sales | Blue-gray |
| Purchasing | Green-gray |
| Journal / Accounting | Amber-gray |
| Inventory | Purple-gray |
| Banking / Payments | Teal-gray |
| Neutral / System | Gray |

Use family tint to identify the workspace, not to indicate business status.

## Rules

- Do not use multiple saturated colors simultaneously.
- Do not use navigation color as status color.
- Status badges must rely on semantic state colors.
- Text contrast must meet WCAG AA expectations for normal enterprise UI text.
- Dark mode readiness requires every new color to have an equivalent token path.
