# VAT Dashboard

## 1. Module Overview
The VAT Dashboard serves as the central hub for monitoring and managing compliance related to this module. It provides a high-level view of tax obligations, book generations, and CAS requirements. Data is heavily dependent on accurate Setup configurations and posted Journal Entries.

## 2. Dashboard UI
### The Action Bar
* **Search:** Find specific records by reference or period.
* **Filter:** Filter by Date Range, Status, or Type.
* **Export:** Download to Excel/CSV for BIR submission or offline review.
* **Import:** Import external adjustments or offline templates.
* **Generate:** Pull latest data from Accounting ledgers.

### The Data List
Displays generated records with:
* Reference / Period
* Total Amounts
* Status (Draft, Final, Filed)
* Actions (View, Edit, Print)

## 3. Data Fields
### Header Information
| Field Name | Description | Required? | UI Component |
| :--- | :--- | :--- | :--- |
| Period | The reporting period (Month/Quarter/Year). | Yes | Date Picker |
| Status | Current status (e.g., Pending, Generated, Filed). | Yes | Dropdown linked to Status |
| Total Amount | Aggregate amount for the period. Auto-filled from upstream document. | Yes | Read-only computed field |

## 4. Supabase Database Architecture
Data is structured into Header and Line-Item tables to maintain proper normalization and support BIR CAS standards.

**Critical Database Rule for Lovable:** A Unique constraint applies to `company_id` + `period` to prevent duplicate reports for the same period.

### Table 1: `compliance_vat_dashboard_headers`
| Column Name | Data Type | Rules | What it stores |
| :--- | :--- | :--- | :--- |
| `id` | UUID | Primary Key | System ID. |
| `company_id` | UUID | Required, Foreign Key | Links to Setup module. |
| `period` | Date | Required | Reporting period. |
| `status` | Text | Required | Draft, Final. |
| `created_by` | UUID | Required, Foreign Key | User ID. |
| `updated_by` | UUID | Nullable, Foreign Key | User ID. |
| `created_at` | Timestamp | Auto | Date created. |
| `updated_at` | Timestamp | Auto | Date updated. |

### Table 2: `compliance_vat_dashboard_lines`
| Column Name | Data Type | Rules | What it stores |
| :--- | :--- | :--- | :--- |
| `id` | UUID | Primary Key | System ID. |
| `header_id` | UUID | Required, Foreign Key | Links to `compliance_vat_dashboard_headers`. |
| `transaction_id` | UUID | Required, Foreign Key | Direct link to Accounting Ledger entry. |
| `amount` | Numeric | Required | Item amount. |
| `created_by` | UUID | Required, Foreign Key | Identifies the user who created the record. |
| `updated_by` | UUID | Nullable, Foreign Key | Identifies the user who last updated the record. |
| `created_at` | Timestamp | Auto | Date created. |
| `updated_at` | Timestamp | Auto | Date updated. |
